# Connect Four AI

A web-based Connect Four game built using **Python, Flask, NumPy, HTML, CSS, and JavaScript**. The game features three AI difficulty levels, including a smart AI powered by the **Minimax Algorithm with Alpha-Beta Pruning**.

## Features

* Easy, Medium, and Hard AI modes
* Intelligent move selection using Minimax
* Win and draw detection
* Interactive web interface

## Technologies

* Python
* Flask
* NumPy
* HTML, CSS, JavaScript

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open:

```text
https://connectfourai.onrender.com
```

## Deployment (Render)

**Build Command**

```bash
pip install -r requirements.txt
```

**Start Command**

```bash
gunicorn app:app
```
